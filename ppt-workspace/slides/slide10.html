04 主要功能介绍

用例编排 - 可视化流程设计

基于 React Flow 的可视化画布

通过拖拽式操作将不同类型的节点组合为完整的测试流程,支持节点间连线、参数传递和变量提取,大幅降低用例编排的技术门槛。

节点类型

Start 节点 - 流程起始节点

API 节点 - 调用API接口,支持参数化配置

Wait 节点 - 等待指定时间后继续执行

Assertion 节点 - 对响应进行断言验证

Parallel 节点 - 并行分支执行多个步骤

End 节点 - 流程结束节点

编排能力

参数化配置

Headers / Query / Path / Body 参数灵活配置

变量提取与传递

从响应中提取变量,供后续步骤引用

断言验证

状态码 / 响应体 / Header / JSONPath 断言

优先级管理

P0-P3 四级优先级,灵活管理用例执行策略
